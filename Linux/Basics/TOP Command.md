```
# top
c -> to show command absolute path
k -> kill the process by PID
n -> to change the no.of task displayed
d,s -> to change interval of refresh
M -> to sort the Linux process by memory usage
m -> toggle memory usage
r -> to change nice value of the PID
u -> to filter task by user
f -> field management
h -> help
z or b -> toggle 'z' color/mono; b -> bold 
x -> sort fields
y -> running tasks
```
